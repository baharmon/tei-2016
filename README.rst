 Build and publish
 -----------------

 These pages are built from the source
 code in a separate directory and published using a script.

 In your clone of the repository in the master branch
 use the `build.sh` script to build the pages::

         ./build.sh

 The directory where pages are built is inside of the directory with
 the source code and is named `build`.
 You can look at the pages using::

         chrome build/index.html

 Commit and push changes to the repository
 when publishing so that changes are shared with other contributors.
 After committing and pushing use the script `publish.sh`
 to publish the updated pages.
